# CIPMS Citizen Portal & Command Dashboard

A fast React Single Page Application configured with Tailwind CSS, TypeScript, and Leaflet Maps.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment Paths**:
   Create a `.env` file in this directory to register backend routing endpoints:
   ```env
   VITE_API_URL=http://localhost:5000
   ```

3. **Run Dev server**:
   ```bash
   npm run dev
   ```

4. **Verify Application build**:
   ```bash
   npm run build
   ```
